### Установка:

1. Выполняем git clone в консоле:
~~~
$ git clone https://github.com/JOKER-THE/golang-todolist.git
~~~

2. Переходим в каталог проекта.

3. Выполняем в консоле сборку и запуск проекта:
~~~
$ go run index.go
~~~

4. Открываем сайт через браузер:
~~~
http://localhost:8181
~~~
